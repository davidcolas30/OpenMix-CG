# Figuras de arquitectura

Esta carpeta conserva una copia renderizada de los diagramas Mermaid principales
de la documentación técnica.

La fuente principal sigue estando en los bloques Mermaid incluidos en
`Documentacion/Arquitectura/` y `Documentacion/Glosario.md`. Los archivos de
esta carpeta son una salida de apoyo para revisar, reutilizar o insertar las
figuras en documentos externos.

## Estructura

- `Mermaid/mmd/`: fuentes Mermaid finales.
- `Mermaid/svg/`: versiones vectoriales.
- `Mermaid/png/`: versiones raster.

Algunas figuras incluyen el sufijo `word-layout` porque se ajustaron para
formatos de página estrechos sin cambiar el contenido técnico del diagrama.
